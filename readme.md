# SDC Prime Number Finder

This project is a tool for finding prime numbers within a given dataset. 
It processes Excel files and identifies all prime numbers contained within the specified data range. 
The tool is designed to be efficient and easy to use, providing both standard
and optional plain text outputs for user convenience.

Run example (or you your data set): 
java -jar target/SDC-1.0-SNAPSHOT.jar sample-data/vzorek_dat.xlsx

Optional plain output (or use your data set):
java -jar target/SDC-1.0-SNAPSHOT.jar sample-data/vzorek_dat.xlsx true
